I built a search engine for MCPs.

There are thousands of MCPs out there. And every week there are more.

The problem? You never know if one already exists for your use case , so you either waste time building something yourself, or you just... don't bother.

So I indexed 4,764 MCPs into a search engine you can query in plain language.

You describe what you want to do → it shows you the MCPs that can do it.

→ Type anything: "automate my calendar", "post to LinkedIn", "query my database"
→ See the exact capabilities that match (each MCP exposes "tools" meaning the specific actions it can perform)
→ Open any result for an AI-generated explanation of what it actually does
→ Filter by category, sort by popularity or quality
